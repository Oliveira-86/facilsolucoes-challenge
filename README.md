## Rodar back end

### Fazer build do back end

    $ ./mvnw clean install -DskipTests 

### Rodar back end 

    $ ./mvnw spring-boot:run

## Fazer build da aplicacao

    $ npm install

## Rodar a aplicacao

    $ yarn start
